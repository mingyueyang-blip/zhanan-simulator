# 时间管理大师 ⏰

一个在手机壳里完成聊天、打工、送礼和多线关系经营的 AI 驱动 demo。

## 功能

- 💬 与多个角色 AI 聊天
- 💰 打工赚钱（送外卖、写评论、老虎机）
- 🛍️ 购物送礼提升好感度
- 📱 逼真的手机 UI 界面

## 技术栈

React + TypeScript + Vite + Tailwind CSS + Zustand + Framer Motion

## 开发

```bash
npm install
npm run dev
```

## 部署

本项目通过 GitHub Pages 自动部署，推送到 `main` 分支即可触发。
